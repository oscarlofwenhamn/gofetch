# gofetch - minimal neofetch clone in go

Powered by BubbleTea, with the purpose of language- and library practice

## Features

- Default view:
  - ASCII icon
  - hostname
  - OS
  - Kernel
  - Uptime
  - Packages
  - Shell (+ version)
  - Theme
  - Icons
  - Terminal
  - CPU
  - GPU
  - Memory (current/total)
  - Color scheme
- Live view
  - Live view, live version
- ...
